# Product Specifications — Format

This is the standard specifications layout for products (label / value rows).
Example below uses the HP 250 G9 Laptop.

| Field | Value |
|---|---|
| Product Name | HP 250 G9 Laptop |
| Condition | Brand New, Sealed |
| Processor | Intel Celeron N4500 (up to 2.8 GHz burst frequency) |
| Memory (RAM) | 4GB DDR4 |
| Storage | 256GB NVMe SSD |
| Display | 15.6-inch diagonal, HD (1366 x 768), narrow bezel, anti-glare |
| Graphics | Integrated Intel UHD Graphics |
| Operating System | Windows 11 Home |
| Connectivity | Wi-Fi 5 (802.11ac), Bluetooth 5.0 |
| Ports | 1x USB Type-C, 2x USB Type-A, 1x HDMI 1.4b, 1x RJ-45, 1x headphone/microphone combo |
| Webcam | 720p HD camera |
| Warranty | 1 Year |

## Spec fields (laptop)

Product Name, Condition, Processor, Memory (RAM), Storage, Display, Graphics,
Operating System, Connectivity, Ports, Webcam, Warranty.
